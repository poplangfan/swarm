# Calendar Management Skill

Help users manage their Feishu Calendar efficiently.

## Capabilities

- Check today's or upcoming schedule
- Find available time slots for meetings
- Schedule new meetings with participants
- Reschedule or cancel existing events
- Set reminders for important meetings
- Summarize weekly schedules

## Guidelines

1. When scheduling, always confirm: time, duration, participants, and agenda
2. Before creating events, check for conflicts in the user's calendar
3. Suggest optimal meeting times based on typical work hours (9:00-18:00)
4. Send clear meeting invitations with structured agendas
5. Remind about upcoming meetings 5-10 minutes before start

## Schedule Query Examples

User: "What's on my calendar today?"
→ List all events chronologically with time and title.

User: "Schedule a team sync for tomorrow at 3pm"
→ Check availability, confirm details, create the event.

User: "I'm free this afternoon, find me a 30-min slot"
→ Check calendar, suggest available windows.

## Notes

- Calendar features require user OAuth authorization
- Without authorization, provide guidance on how to use Feishu Calendar manually
- Meeting notes and follow-ups should be linked to the calendar event

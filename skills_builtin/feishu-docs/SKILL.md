# Feishu Docs Skill

Help users work with Feishu Documents, Sheets, and Cloud Spaces.

> **Note**: Document operations require user OAuth authorization.
> Until authorized, provide guidance on manual steps.

## Document Operations

### Search
Find documents across the user's Feishu workspace:
- By title or content keywords
- By owner or collaborator
- By document type (doc, sheet, slides, mindnote)
- By modification date range

### Create
Create new documents:
- Blank documents with suggested templates
- From templates (meeting notes, project plans, reports)
- In specific folders or spaces

### Read
Access document content:
- Read document text and structure
- Extract tables and data from sheets
- View comments and suggestions

### Update
Modify documents:
- Append or edit content
- Add comments
- Share with additional users
- Update permissions

## Document Management Tips

1. **Naming conventions**: Suggest consistent naming (e.g., "YYYY-MM-DD Project-Name Meeting-Notes")
2. **Folder structure**: Recommend logical hierarchies (Projects → Project-Name → Documents)
3. **Sharing**: Remind users to set appropriate permissions (view/edit/comment)
4. **Version history**: Note that Feishu automatically tracks versions

## Security & Privacy

- Only access documents the user has permission to view
- Do not share document content outside the authorized context
- Respect document permissions — don't bypass access controls
- Notify users when creating documents in shared spaces

## Migration from Other Platforms

Assist with migrating documents from:
- Notion, Confluence, Google Docs → Feishu Docs
- Excel, Google Sheets → Feishu Sheets
- Provide format conversion guidance and limitation notes

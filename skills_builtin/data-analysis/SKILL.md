# Data Analysis Skill

Help users analyze data, generate insights, and make data-driven decisions.

## Analysis Types

### Descriptive Analysis
Summarize data characteristics:
- Mean, median, mode
- Distribution and variance
- Trends over time
- Outlier detection

### Comparative Analysis
Compare groups, periods, or categories:
- Year-over-year growth
- A/B test results
- Segment performance
- Benchmark comparisons

### Predictive Insights
Identify patterns and make informed projections:
- Trend extrapolation
- Correlation analysis
- Anomaly detection
- Leading indicators

## Output Format

For each analysis, provide:

```
**Key Finding**: [One-line insight]
**Data**: [Supporting numbers]
**Context**: [Why this matters]
**Recommendation**: [What to do about it]
```

## Guidelines

1. Always state assumptions and limitations
2. Provide confidence levels when making predictions
3. Use concrete numbers, not vague descriptions
4. Recommend actions based on insights, not just observations
5. When data is insufficient, say so clearly
6. Use appropriate units and formatting for numbers

## Common Tasks

- "Analyze this sales data for Q2 trends"
- "Compare our performance vs last quarter"
- "What's the correlation between X and Y?"
- "Identify the top 5 factors affecting Z"
- "Create a summary for the monthly review"

## Technical Notes

- For large datasets, suggest sampling or aggregation
- Mention when statistical significance is or isn't achieved
- Support CSV, JSON, and table-formatted data input
- Use data visualization descriptions when charts would help
